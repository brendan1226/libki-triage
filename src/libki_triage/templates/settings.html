GitHub integration

Configure your personal GitHub token so PRs are created under your account. The token needs repo scope (for forking, branching, committing, and opening PRs).

How to create a GitHub token

  1. Go to GitHub → Settings → Personal Access Tokens → Fine-grained
  2. Click "Generate new token"
  3. Name: libki-triage
  4. Repository access: "All repositories" or select the Libki repos
  5. Permissions: Contents: Read and write, Pull requests: Read and write
  6. Generate → copy the token → paste above
